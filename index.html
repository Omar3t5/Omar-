⬅ الرجوع للرئيسية

البرمجة هي كتابة تعليمات منطقية للحاسوب لحل مشكلة أو تنفيذ مهمة. فيما يلي أساسيات يجب معرفتها كبداية.

1) الخوارزميات Algorithms

ابدأ
1) اغلي الماء
2) ضع الشاي والسكر
3) صب الماء وحرك
4) اشرب الشاي
انتهى

2) المتغيرات Variables

// JavaScript
let age = 20;
age = age + 1; // أصبحت 21

3) أنواع البيانات Data Types

  • Number — الأرقام
  • String — النصوص مثل "Hello"
  • Boolean — منطقي (true/false)
  • Array — مصفوفة: قائمة قيم
  • Object — كائن: مفاتيح وقيم

4) الشروط Conditionals

// JavaScript
const temp = 30;
if (temp > 25) {
  console.log("الجو حار");
} else {
  console.log("الجو لطيف");
}

5) الحلقات Loops

// for
for (let i = 1; i <= 3; i++) {
  console.log(i);
}
// while
let n = 3;
while (n > 0) { console.log(n); n--; }

6) الدوال Functions

function greet(name) {
  return `مرحبا ${name}!`;
}
console.log(greet("Omar"));
